Conference paper

Automatic License Plate Recognition System For Bangladeshi Vehicles Using Deep Neural Network

This paper was awarded as the best paper at the conference in Machine Learning track.

This paper has proposed an optimal end-to-end approach for the Automatic License Plate Recognition(ALPR) system for Bangladeshi vehicles by experimenting with the various Deep Neural Network(DNN) models. These models have been trained and evaluated on our rich datasets of Bangladeshi vehicles and license plates. We have also introduced an algorithm that eliminates the need for the typical segmentation phase and generates properly formatted output efficiently. The final proposed system offers 99.37% accuracy in license plate localization and 96.31% accuracy in text recognition from the license plate(LP)s.

Automatic License Plate Recognizer

Python YOLO OpenCV

An AI system that can automatically detect Bangladeshi license plates and then recognize what is written on them. After that, the plate number with some additional information(time/date) is written into a CSV file. We have used a pre-trained model to detect and recognize. We have made several version with the combination of YoloV4 and Tensorflow object detection model (such as:EfficientDetD0, EfficientDetD1,EfficientDet2, SSD MobileNet V2 FPNLite 320x320 , SSD MobileNet V1 FPN 640x640, SSD ResNet50 V1 FPN 640x640 (RetinaNet50)). Later we used OpenCV and TensorFlow object detection API to make our desired output. It is written in python. This was made for my varsity project.
